# 준비된 면접관

> 좋은 채용은 우연이 아니다.
> 준비된 면접관이 만들어내는 결과다.

면접관이 갖춰야 할 관점, 시스템, 질문 기법,
평가 방법까지 단계별로 다루는 실전 가이드입니다.

---

## 📚 전체 구성

총 **4부 16장**으로 구성됩니다.

- **1부**. 면접관의 관점 — 왜 면접이 중요한가
- **2부**. 채용을 시스템으로 만들기
- **3부**. 역량 기반 면접
- **4부**. 면접 실전과 면접관의 성장

---

## 📑 목차

### 1부. 면접관의 관점

면접관이 가져야 할 근본적인 관점과 자세를 다룹니다.

- [1장. 사람이 곧 기업의 성과다](01장.md)
  세계적 기업들이 채용에 시간과 자원을 쏟는 이유

- [2장. 잘못된 채용의 진짜 비용](02장.md)
  보이는 비용과 보이지 않는 비용, 그리고 장기 영향

- [3장. 면접은 갑과 을이 없다](03장.md)
  상호 평가의 원칙, 면접관이 회사의 얼굴인 이유

- [4장. 면접관이 빠지기 쉬운 평가 오류](04장.md)
  후광·낙인·첫인상·유사성·말솜씨 오류와 자기 점검법

---

### 2부. 채용을 시스템으로 만들기

좋은 채용을 운에 맡기지 않고 시스템화하는 방법을 다룹니다.

- [5장. 채용 실패 사례에서 배운다](05장.md)
  실패 사례에서 공통적으로 발견되는 패턴

- [6장. 좋은 채용 프로세스의 조건](06장.md)
  적합성, 검증 도구, 면접관 전문성, 시스템화

- [7장. 면접 외 검증 도구 활용법](07장.md)

- [8장. 타당도와 신뢰도로 점검하기](08장.md)
  채용 프로세스 자체를 점검·개선하는 기준

### 3부. 역량 기반 면접

- [9장. 역량이라는 개념](09장.md)

- [10장. 행동사례 면접과 STAR 기법](10장.md)

### 4부. 면접 실전과 면접관의 성장

질문 설계부터 평가, 그리고 면접관 시스템화까지 다룹니다.

- [13장. 질문 설계와 유형](13장.md)
  직접·과거 행위·상황 질문의 사용법과 좋은 질문의 조건

- [14장. 면접 형태와 진행](14장.md)

- [15장. 평가와 의사결정](15장.md)
  평가표 작성, 면접관 간 의견 조율, 합·불 결정
