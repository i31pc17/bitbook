# '원온원', 요즘 시대 리더십 필수품

요즘 들어 원온원을 학습하는 기업과 리더가 정말 많아졌습니다.

처음 《원온원》이 나왔을 때는  
스타트업 중심의 리더십이었습니다.

지금은 주요 대기업뿐 아니라  
중소기업도 원온원을 중요하게 여기며  
학습과 확산에 힘쓰고 있습니다.

이유는 간단합니다.

회의에서 말하지 못하는 이야기를  
나눌 수 있기 때문입니다.

> 90분을 일대일로 소통하면  
> 팀원의 업무 퀄리티를 2주, 80시간 이상 높일 수 있다.  
> — 《하이 아웃풋 매니지먼트》

> 짧은 대화 몇 번으로 원온원을 대체할 수 없다.  
> 구성원은 저마다 니즈와 경험이 다르고,  
> 높은 성과를 내는 리더는 이를 잘 안다.  
> — 앤디 그로브

미국이니까 가능하다는 의견도 있지만,  
우리나라 리더들도 비슷하게 말합니다.

* 대면으로 하니 업무 이야기를 더 솔직하게 들을 수 있었다.
* 팀원이 마음을 열어주어, 나도 몰랐던 부분을 배웠다.
* 팀원 10명과 매달 대화하니 내가 가장 많이 배운다.

---

## 모든 세대의 1위가 원온원이었다

한 연구소가 1970·1980·1990년대생  
각 200명씩 총 600명에게  
합리적인 소통 방식을 물었습니다.

2·3위는 세대마다 달랐지만,  
1위는 모두 원온원이었습니다.

> 90년대생 34%, 80년대생 35%, 70년대생 35.5%

원온원의 핵심은  
정기적으로 팀원의 이야기를 듣는 경청의 시간입니다.

리더보다 팀원이 더 많이 말할 수 있어야 합니다.

그렇게 소통한 뒤,  
탁월한 리더는 어떤 리더이고  
함께 일하고 싶은 리더는 어떤 행동을 반복하는지  
나만의 답을 정리해보면 좋습니다.

---

## 원온원을 조직의 리더십으로 만드는 법

말로는 쉬운 원온원을  
처음 시작할 때 많은 리더가 힘들어합니다.

* "술 없이 어떻게 대화하죠? 그래본 적이 없어요."
* "마주 앉기만 해도 땀이 나고, 머리가 하얗게 됩니다."

원인은 하나입니다.  
팀장도 원온원을 받아본 적이 없다는 것입니다.

경험해보지 못한 팀장에게  
배워서 실행하라고만 해서는 적용되지 않습니다.

원온원을 조직의 리더십으로 정착시킨 기업들에는  
여섯 가지 공통된 장치가 있었습니다.

---

### 첫째, 리더십이 아니라 조직문화로 본다

> 원온원은 리더십이 아니라 문화입니다.

처음에는 원온원을 리더십으로 보고  
리더들이 배우면 된다고 믿었습니다.

하지만 변하지 않으려는 팀원,  
출퇴근 시간만 중시하는 팀원과의 원온원에서  
벽을 마주한 팀장들을 보며  
이것이 조직문화임을 인정하게 됐습니다.

원온원이 활발한 조직은  
리더보다 팀원이 원온원을 더 중요하게 여깁니다.

다음 유형의 팀원이 많은 회사가 빠르게 정착했습니다.

* 자신의 일을 자랑하고 확산하고 싶은 직원
* 커리어에 높은 꿈과 비전이 있는 직원
* 주어진 일을 더 잘하고 싶은 욕심이 있는 직원
* 공부하고 학습하기를 좋아하는 직원

내향형보다 외향형이 더 잘한다는 가설은 맞지 않았습니다.

더 잘하고 싶은 마음과 더 배우고 싶은 열정,  
이 두 가지면 충분했습니다.

---

### 둘째, 리더의 역할과 평가 기준을 재정의한다

2021년부터 원온원을 도입한 A 기업은  
가장 먼저 리더십 평가 기준을 바꿨습니다.

이전 기준은 성과 90%, 역량 10%였습니다.

성과만 잘 내면 좋은 평가를 받고  
성과 좋은 팀원이 팀장이 되는 구조라,  
리더가 팀원의 성장에 관심을 둘 이유가 없었습니다.

원온원을 도입하며  
평가 기준을 성과 50%, 성장 50%로 바꿨습니다.

그러자 리더들이 먼저  
어떻게 팀원의 성장을 도울지 묻기 시작했습니다.

움직이지 않던 리더를 움직인 것은  
역할을 '성과'에서 '성과와 성장'으로 바꾸고,  
거기 맞는 평가 기준을 새로 만든 것이었습니다.

---

### 셋째, 팀장의 성장을 돕는 시스템을 둔다

많은 기업이 팀장을 통해 원온원을 확산하면서도,  
정작 팀장의 성장과 성공을 돕는 제도는 없습니다.

팀장도 동기부여와 도움이 필요합니다.

* 고민이 있을 때 코칭받을 사내·사외 코치
* 리더십을 멘토링해줄 선배 팀장

상사에게 말 못 할 고민,  
팀원에게 어떤 리더십을 보여야 할지에 대한 고민을  
전문 코치를 통해 실시간으로 묻고 학습하는 것입니다.

---

### 넷째, 스킬셋을 꾸준히 학습한다

스킬 교육도 중요하지만,  
리더십 재정의·조직문화·시스템이 먼저고  
스킬은 그다음입니다.

학습하면 좋은 주제는 다음과 같습니다.

1. **질문** — 상대의 생각을 확장하는 질문
2. **중립 대화** — 평가와 판단을 하지 않는 대화
3. **경청** — 상대가 마음을 열게 하는 듣기
4. **심리적 안전감** — 무슨 말을 해도 안전하다는 믿음 주기
5. **수평적 조직** — 더 나은 대안을 찾는 팀 문화
6. **피드백** — 평가보다 피드백·피드포워드
7. **인정·칭찬** — 인정과 칭찬으로 동기부여
8. **학습 문화** — 고민·지식·경험을 공유하는 팀 문화
9. **사람 이해** — MBTI·에니어그램·DISC 등으로 서로의 차이 이해
10. **일의 의미** — 내 일과 팀원 일의 의미와 영향 찾기

---

### 다섯째, 매뉴얼이 있어야 한다

스킬셋의 핵심은 배우는 것이 아니라  
현장에서 쓰는 것입니다.

그래서 어떤 기업은  
팀장이 자리에서 꺼내볼 간단한 매뉴얼을 만들어둡니다.

신입 배치, 성과관리, 성장·커리어 대화,  
팀원 갈등, 행동 변화가 없을 때 등  
상황별 질문·경청법·사례를 정리한 것입니다.

A 기업은 한 해의 월별 주제를 정해두었는데,  
팀장 주도와 팀원 주도 대화를 구분한 것이 특징입니다.

* **1월** (리더 주도) — 전년도 성과평가 결과 공유, 격려
* **2월** (리더 주도) — 회사·팀·개인 목표 수립과 얼라인
* **4·7·10월** (리더 주도) — 분기 중간 피드백
* **3·5·6·8·9·11월** (팀원 주도) — 상시 성과관리
* **12월** (리더 주도) — 연간 성과평가 피드백

리더 주도 달에도  
대화 시간의 70~80%만 팀장 주제로 쓰고,  
나머지는 팀원이 나누고 싶은 주제로 마무리합니다.

팀원 주도 달에는  
"오늘은 어떤 주제로 이야기하고 싶나요?"로 시작합니다.

이를 위해 매주 자신의 과업과 궁금한 점을  
메모해두는 습관이 생긴 팀원도 있었습니다.

#### PDP — 개인 성장 플랜

7~9월에는 팀원이 원하는 시기에  
커리어 원온원을 따로 진행합니다.

이때 쓰는 도구가  
`PDP`(Personal Development Plan)입니다.

과거·현재의 커리어와 그동안 쌓은 지식·경험을 적고,  
3년·5년 후의 모습과 필요한 경험·학습·브랜딩 방법을  
리더와 함께 설계하는 표입니다.

| 시점 | 직급·직책 | 부서·과업 | 확보할 경험 | 브랜딩 방법 | 필요한 교육 |
|---|---|---|---|---|---|
| 5년 후 | | | | | |
| 3년 후 | | | | | |
| 현재 | | | | | |
| 과거 | | | | | |

1년에 한 번 작년 PDP를 피드백하고 다시 설계하며  
팀원의 성장을 돕습니다.

---

### 여섯째, 단계별로 진행한다

팀장이 원온원을 힘들어하는 가장 큰 이유는  
팀장 자신이 원온원을 받아본 적이 없기 때문입니다.

조직은 성장보다 성과를 중시해왔고,  
가정에서도 부모와 편안히 대화한 경험이 많지 않습니다.

이미 원온원을 잘하는 리더들의 공통점은  
성별·나이·기업 규모가 아니었습니다.

어릴 때부터 아버지와 토론하고 대화한 리더,  
그다음이 코칭 리더십을 학습한 리더였습니다.

스스로 좋은 경험이 없다면  
팀원에게 좋은 경험을 전하기 어렵습니다.

그래서 가장 좋은 방법은  
CEO가 임원과, 임원이 팀장과 원온원을 하는  
위에서부터의 문화입니다.

원온원을 가장 많이 학습해야 할 순서는 이렇습니다.

> CEO·HR → 임원 → 팀장 → 팀원

HR과 임원이 팀장과 원온원을 하며  
그들의 고민과 성장을 돕는 문화가 있어야 합니다.

---

## 기억할 포인트

* 세대를 막론하고 가장 합리적인 소통 1위는 원온원이었다.
* 원온원은 리더십이기 전에 조직문화다.
* 역할과 평가 기준을 성과에서 '성과와 성장'으로 재정의해야 움직인다.
* 팀장의 성장을 돕는 코치와 시스템이 있어야 한다.
* 스킬보다 리더십 재정의·문화·시스템이 먼저다.
* 매뉴얼과 PDP로 배운 것을 현장에서 쓰게 한다.
* CEO·임원이 먼저 원온원을 경험하는 위에서부터의 문화가 필요하다.
